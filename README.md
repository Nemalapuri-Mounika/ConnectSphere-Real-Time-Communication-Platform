# ConnectSphere – Real-Time Chat Application

## Overview
ConnectSphere is a full-stack real-time chat application developed to simulate a production-level messaging system. It enables users to communicate instantly using WebSockets, with secure authentication, private messaging, and persistent chat history.

---

## Tech Stack
- Frontend: React.js  
- Backend: Node.js, Express.js  
- Real-time: Socket.IO  
- Database: MongoDB  
- Authentication: JWT  

---

## Features
- Secure User Authentication (JWT)
- Real-time messaging using WebSockets
- One-to-one private chat
- Online/offline user status
- Typing indicator
- Persistent chat history (database)
